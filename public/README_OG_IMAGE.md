# Open Graph Image

Для корректного отображения превью в социальных сетях (Telegram, VK, Facebook) необходимо создать файл:

**og-image.jpg**

## Требования:

- **Размер:** 1200x630 пикселей (рекомендуемый)
- **Формат:** JPG или PNG
- **Содержание:** 
  - Логотип N2O Rostov
  - Текст: "Закись азота N2O в Ростове"
  - Подзаголовок: "Доставка 30-60 минут"
  - Золотой цвет (#D4AF37) на темном фоне

## Размещение:

Файл должен быть доступен по адресу:
`https://azotrostovskiy.ru/og-image.jpg`

После создания загрузите файл в папку `public/` и пересоберите проект.

## Генерация:

Можно использовать:
- Canva (https://canva.com) - шаблон 1200x630
- Figma
- Photoshop
