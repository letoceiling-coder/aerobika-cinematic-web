# Favicon Files

Для production сайта необходимо создать следующие файлы favicon:

1. **favicon.ico** - основной favicon (16x16, 32x32, 48x48)
2. **favicon-32x32.png** - PNG favicon 32x32
3. **favicon-16x16.png** - PNG favicon 16x16
4. **apple-touch-icon.png** - Apple touch icon 180x180

## Рекомендации:

- Используйте логотип N2O или баллон закиси азота
- Цвета: золотой (#D4AF37) на темном фоне
- Формат: PNG для всех, кроме favicon.ico (ICO)
- Размеры: точно указанные выше

## Генерация:

Можно использовать онлайн генераторы:
- https://realfavicongenerator.net/
- https://favicon.io/

Загрузите готовые файлы в папку `public/`
